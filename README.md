# Front-end Java de gestão de vagas
## Tecnologias
+ Spring Boot - Framework.
+ Spring Devtools - Quick reload.
+ Spring Web
+ TailwindCSS - estilização com CDN do tailwind.
+ Thymeleaf - frontend dinâmico.
## Como rodar a aplicação
### Rodar no terminal
```bash
mvn spring:boot:run
```
